# Complete Web Application (Django)

This is a simple Django web application developed as Project 3.

## Features
- Add items with name and description
- View added items
- Django MVC structure
- SQLite database

## How to Run
1. Install Django
   pip install django
2. Run migrations
   python manage.py migrate
3. Start server
   python manage.py runserver

## Tech Stack
- Python
- Django
- HTML
- SQLite
